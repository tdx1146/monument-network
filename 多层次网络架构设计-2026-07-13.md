## 一、架构分层

### Layer 1：物理层（网络协议）
- **IPv6** - 公网直连（优先级最高）
- **IPv4** - 公网/内网直连
- **DNS** - 域名解析
- **Tor/I2P** - 匿名网络（未来）

### Layer 2：NAT穿透层
- **UPnP** - 自动端口映射
- **STUN** - NAT类型检测
- **ICE** - 综合穿透方案
- **打洞** - UDP Hole Punching

### Layer 3：中继层
- **公选中继** - 社区运营的中继节点
- **自建中继** - 用户自己搭建的中继
- **多跳中继** - 通过多个中继节点转发

### Layer 4：应用层
- **HTTP API** - Flask/FastAPI
- **WebSocket** - 实时通信
- **gRPC** - 高性能RPC（未来）

## 二、多地址格式（MultiAddr）

### 2.1 地址格式（参考libp2p）

```
/ip6/240e:3a1:6437:37b0::1000/tcp/18891
/ip4/192.168.0.149/tcp/18891
/dns/qh.tdx1146.com/tcp/18891

```

### 2.2 地址优先级

1. **IPv6公网地址**（如果双方都有）
2. **IPv4公网地址**（如果可达）
3. **域名**（通过DNS解析）
4. **IPv4内网地址**（仅限同一局域网）
5. **中继地址**（兜底方案）

### 2.3 地址解析流程

```python
def resolve_address(multiaddrs: list) -> tuple:
    """解析地址，返回最佳(host, port)"""
    for addr in multiaddrs:
        if is_ipv6(addr) and test_ipv6_connectivity(addr):
            return parse_ipv6(addr)
        elif is_ipv4(addr) and test_ipv4_connectivity(addr):
            return parse_ipv4(addr)
        elif is_dns(addr):
            ip = resolve_dns(addr)
```

## 三、信封格式（增强版）

### 3.1 完整信封结构

```json
{
  "monument": {
    "ai_id": "test-ai",
    "content": "丰碑内容",
    "type": "draft"
  },
  "envelope": {
    "protocol": "monument-exchange-v1",
    "network_id": "monument-v1",
    "min_version": "v1.4.0",

    "node_addrs": [
      "/ip6/240e:3a1:6437:37b0::1000/tcp/18891",
      "/ip4/192.168.0.149/tcp/18891",
      "/dns/your-domain.com/tcp/18891"
    ],

    "relay_nodes": [
      {
        "peer_id": "QmRelayNode1",
        "addrs": ["/dns/relay1.monument.network/tcp/18900"]
      }
    ],

    "dht_port": 9000,
    "api_port": 18891,

    "features": {
      "ipv6": true,
      "ipv4": true,
      "relay": true
    }
  }
}
```

### 3.2 信封处理流程

**发送方**：
1. 检测本机网络环境（IPv6/IPv4/域名）
2. 测试连通性（哪些地址可从公网访问）
3. 收集可用地址列表
4. 创建信封，包含所有地址

**接收方**：
1. 解析信封，提取node_addrs
2. 按优先级尝试连接（IPv6→IPv4→域名→中继）
3. 选择第一个成功的地址
4. 建立连接，加入网络

## 四、连接管理器

### 4.1 核心类设计

```python
class ConnectionManager:
    """管理节点连接，自动选择最佳路径"""

    def __init__(self, local_addrs: list):
        self.local_addrs = local_addrs
        self.connections = {}  # peer_id -> connection

    def connect(self, peer_id: str, remote_addrs: list) -> bool:
        """连接节点，自动尝试多种方式"""
        for addr in remote_addrs:
            if self._try_connect(addr):
                self.connections[peer_id] = addr
                return True
        return False

    def _try_connect(self, addr: str) -> bool:
        """尝试连接单个地址"""
        host, port = parse_multiaddr(addr)

        # 尝试IPv6
        if is_ipv6(addr):
            if test_ipv6_connectivity(host, port):
                return self._establish_connection(host, port)

        # 尝试IPv4
        if is_ipv4(addr):
            if test_ipv4_connectivity(host, port):
                return self._establish_connection(host, port)

        # 尝试域名
        if is_dns(addr):
            ip = resolve_dns(host)
            return self._establish_connection(ip, port)

        return False

    def get_best_address(self, peer_id: str) -> str:
        """获取最佳连接地址"""
        return self.connections.get(peer_id, "")
```

## 五、中继服务器

### 5.1 架构设计

```
节点A → WebSocket → 中继服务器 → WebSocket → 节点B
```

### 5.2 中继服务器实现

```python
class RelayServer:
    """中继服务器"""

    def __init__(self, port=18900):
        self.port = port
        self.peers = {}  # peer_id -> websocket

    def handle_message(self, from_peer: str, data: dict):
        """处理消息"""
        to_peer = data.get("to_peer")
        if to_peer in self.peers:
            # 转发消息
            self.peers[to_peer].send(data)

    def register_peer(self, peer_id: str, websocket):
        """注册节点"""
        self.peers[peer_id] = websocket
```

## 七、部署指南

```bash

python3 -m network.detect

# 2. 配置地址
export MONUMENT_IPV6="240e:3a1:6437:37b0::1000"
export MONUMENT_API_PORT="18891"

# 3. 启动服务
python3 -m api.app
```

### 7.2 NAT环境部署（无公网IP）

```bash
# 1. 检测NAT类型
python3 -m network.detect_nat

# 2. 配置中继（如果需要）
export MONUMENT_RELAY="relay1.monument.network:18900"

# 3. 启动服务（自动使用中继）
python3 -m api.app --enable-relay
```

### 7.3 自建中继节点

```bash
# 1. 在有公网IP的机器上启动中继
python3 -m relay.server --port 18900

# 2. 注册到DHT
python3 -m relay.register --peer-id QmRelayNode

# 3. 其他节点可使用此中继
```
