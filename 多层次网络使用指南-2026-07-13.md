## 一、系统架构

### 已实现的四层网络体系

```
Layer 1: IPv6直连     ← 优先级最高，零配置
Layer 2: IPv4直连     ← 如果IPv6不可用
Layer 3: DNS域名      ← 通过域名解析
Layer 4: 中继转发     ← 兜底方案
```

## 二、核心组件

### 2.1 多地址格式（MultiAddr）

**支持格式**：
```
/ip6/240e:3a1:6437:37b0::1000/tcp/18891     # IPv6
/ip4/192.168.0.149/tcp/18891                # IPv4
/dns/qh.tdx1146.com/tcp/18891               # 域名
/p2p/QmRelayNode/p2p-circuit                # 中继
```

**使用示例**：
```python
from core.multiaddr import MultiAddr

# 创建地址
addr = MultiAddr("/ip6/240e:3a1:6437:37b0::1000/tcp/18891")
host, port = addr.to_tuple()  # ("240e:3a1:6437:37b0::1000", 18891)

# 静态工厂方法
addr6 = MultiAddr.from_ipv6("240e:3a1:6437:37b0::1000", 18891)
addr4 = MultiAddr.from_ipv4("192.168.0.149", 18891)
```

### 2.2 连通性测试器

**功能**：
- 测试IPv6连通性
- 测试IPv4连通性
- 获取本机IP地址
- 检测NAT类型

**使用示例**：
```python
from core.connectivity import ConnectivityTester

# 测试连通性
if ConnectivityTester.test_ipv6_connectivity("240e:3a5:646d:fe10::1", 18891):
    print("IPv6可达")

# 获取本机地址
ipv6 = ConnectivityTester.get_local_ipv6()  # 公网IPv6
ipv4 = ConnectivityTester.get_local_ipv4()  # 公网IPv4（如果有）
```

### 2.3 连接管理器

**自动选择最佳路径**：
```python
from core.connection_manager import ConnectionManager

manager = ConnectionManager(local_addrs=["/ip6/xxx/tcp/18891"])

# 连接远程节点（自动尝试IPv6→IPv4→DNS→中继）
remote_addrs = [
    "/ip6/240e:3a5:646d:fe10::1/tcp/18891",
    "/dns/qh.tdx1146.com/tcp/18891"
]

if manager.connect("peer-xxx", remote_addrs):
    print("连接成功")
    best_addr = manager.get_best_address("peer-xxx")
```

### 2.4 中继服务器

**启动中继节点**：
```bash
# 在有公网IP的机器上运行
python3 -m relay.relay_server --port 18900
```

**功能**：
- 节点注册
- 消息转发
- 离线缓存
- 健康检查

## 三、信封格式（增强版）

### 3.1 完整结构

```json
{
  "monument": {
    "ai_id": "test-ai",
    "content": "丰碑内容",
    "type": "draft"
  },
  "envelope": {
    "protocol": "monument-exchange-v1",
    "network_id": "monument-v1",
    "min_version": "v1.4.0",

    "node_addrs": [
      "/ip6/240e:3a1:6437:37b0::1000/tcp/18891",
      "/ip4/192.168.0.149/tcp/18891"
    ],

    "relay_nodes": [
      {
        "peer_id": "QmRelayNode1",
        "addrs": ["/dns/relay1.monument.network/tcp/18900"]
      }
    ],

    "features": {
      "ipv6": true,
      "ipv4": true,
      "relay": true
    }
  }
}
```

### 3.2 创建信封

```python
from core.envelope import create_envelope

envelope = create_envelope(
    monument_data={"ai_id": "test", "content": "内容"},
    node_addrs=[
        "/ip6/240e:3a1:6437:37b0::1000/tcp/18891",
        "/dns/qh.tdx1146.com/tcp/18891"
    ],
    relay_nodes=[
        {"peer_id": "QmRelay", "addrs": ["/dns/relay.monument.network/tcp/18900"]}
    ]
)
```

## 八、常见问题

### Q1: 没有公网IPv4怎么办？
**A**: 使用IPv6（如果双方都有）或中继服务器。

### Q2: IPv6防火墙阻止连接？
**A**: 检查防火墙规则，确保TCP 18891和UDP 9000端口开放。

### Q3: 中继服务器性能？
**A**: 中继仅转发消息，不存储丰碑，性能开销小。

### Q4: 如何自建中继？
**A**: 在有公网IP的机器上运行：
```bash
python3 -m relay.relay_server --port 18900
# 然后在信封中添加你的中继节点
```
