### ⚠️ **关键问题：未与丰碑产出咬合**

**问题描述**：
- 玄鉴系统有`write_candidate`触发逻辑
- 但`individual_monument.py`的`write_candidate()`方法**没有调用玄鉴**
- 两者独立运行，没有集成

**影响**：
- 丰碑创建不经过玄鉴评分
- 三轴判别不生效
- 置信度阈值无效

**需要的修复**：
```python
# 在individual_monument.py的write_candidate()中：
def write_candidate(self, content: str, metadata: Optional[dict] = None):
    # Step 1: 玄鉴评分
    from core.xuanjian_pipe import XuanjianPipe
    xuanjian = XuanjianPipe()
    confidence, three_axis = xuanjian.evaluate_insight(content)
    
    # Step 2: 检查置信度
    if confidence < 0.8:
        raise ValueError(f"置信度不足：{confidence} < 0.8")
    
    # Step 3: 写入候选
    entry = {
        "content": content,
        "xuanjian_score": confidence,
        "three_axis": three_axis,
        "created_at": datetime.now()
    }
    self.data["monuments"]["candidates"].append(entry)
```

## 五、丰碑产出流程检查

### 理想流程（设计意图）

```
洞察产生
  ↓
玄鉴评分（三轴判别 + 置信度计算）
  ↓
置信度 >= 0.8 → 写入候选
  ↓
局域层竞争（多维评分）
  ↓
优胜者 → finalized
  ↓
全球广播（签名 + 多副本）
  ↓
全网保存
```

### 实际流程（当前实现）

```
洞察产生
  ↓
手动写入drafts/candidates/finalized  ← ⚠️ 没有玄鉴评分
  ↓
手动广播  ← ⚠️ 没有自动触发
  ↓
局域网/全球网络同步
```

### 缺失的环节

1. **玄鉴评分未集成** - write_candidate()没有调用玄鉴
2. **竞争机制未实现** - 局域层竞争在规划中
3. **自动广播未触发** - finalize后没有自动广播
