### v3.1.0 - 2026-07-16 00:06

**变更类型**：新增
**负责人**：deepseek（子代理）
**影响范围**：
- `core/individual_monument.py`（新增 `write_candidate_scored` 方法）
- `tests/test_individual_monument.py`（新增7个玄鉴集成测试）
- `工作规划.md`（更新进度）

**变更说明**：

**F135：玄鉴嵌入丰碑**

核心改动：

1. **新增 `write_candidate_scored()` 方法**
   - 接收 `xuanjian_pipe` 和 `xuanjian_confidence` 参数
   - 调用 `XuanjianPipe.evaluate()` 进行三轴判别评分
   - 置信度 >= 0.8 → 创建候选丰碑，评分详情写入条目
   - 置信度 < 0.8 → 返回 `(-1, error_info)`，不创建
   - 管道异常 → 返回 `(-1, error_info)`，不创建
   - 无管道（兜底）→ 直接创建，保持兼容

2. **保留原始 `write_candidate()` 接口不变**
   - 所有现有调用者无需修改
   - 向后100%兼容

3. **新增 `get_last_xuanjian_analysis()` 查询方法**
   - 获取最近一次玄鉴分析结果

4. **候选条目增加玄鉴评分信息**
   - `xuanjian_score`：综合置信度
   - `xuanjian_analysis`：完整的三轴判别详情

**备份文件**：
- `core/individual_monument.py.bak`（F135修改前备份）

**回滚方法**：
```bash
cp /vol2/1000/AI专用/丰碑网络/code/core/individual_monument.py.bak /vol2/1000/AI专用/丰碑网络/code/core/individual_monument.py
```

**测试结果**：✅ 18/18 全部通过（11项原始测试 + 7项玄鉴新增）

**测试用例**：
1. `test_write_candidate_scored_high_confidence` — 正常创建（>= 0.8）
2. `test_write_candidate_scored_low_confidence` — 拒绝创建（< 0.8）
3. `test_write_candidate_scored_pipeline_failure` — 评分失败处理
4. `test_write_candidate_scored_no_pipe` — 无管道兜底
5. `test_write_candidate_scored_frozen` — 冻结后阻止
6. `test_write_candidate_scored_after_freeze_raises` — 冻结后异常
